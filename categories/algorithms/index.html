Primality testing

Naive but deterministic

Primes are of great theoretical and practical value in cryptography. A primality test is an algorithm for determining whether an input number is prime. The simplest primality test is trial division: given an input number, n, check whether it is divisible by any prime number between 2 and ‚àön. If so, then n is composite. Otherwise, it is prime.
